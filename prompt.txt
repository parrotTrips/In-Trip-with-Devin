## Projeto: Parrot Trips — In-Trip App

App web para viajantes e staff acompanharem uma experiência de viagem em grupo.
Backend FastAPI (Python), frontend React + TypeScript, banco PostgreSQL no Supabase.

Repositório: https://github.com/parrotTrips/In-Trip-with-Devin
Frontend (produção): https://parrot-trips-app-286.netlify.app
Backend (produção): https://parrot-trips-backend-428743191336.southamerica-east1.run.app

## Dois tipos de usuário

- `role = 'traveler'` → app do viajante (pré-trip com cards de checklist + in-trip com roteiro por dia)
- `role = 'staff'` → staff app (itinerário in-trip, QR scanner, contatos operacionais)

O roteamento acontece em `frontend/src/app/App.tsx` baseado no `user.role`.
Staff tem um botão "Traveler view" no header para ver o app como o viajante veria.

## Viagem de teste

| Campo | Valor |
|-------|-------|
| trip_uuid | `TEST-2026-FULL` |
| Fases pre-trip | 4 (Visa, Vaccinations, Packing, Documents) |
| Dias in-trip | 7 (01–07 Jul 2026) |

Para simular dias no app (muda o modo para in-trip e avança a barra):
```bash
cd backend
poetry run python scripts/simulate_trip_day.py --trip-uuid TEST-2026-FULL --day 1
```

Para resetar do zero (pré-trip, tudo zerado):
```bash
poetry run python scripts/reset_test_trip.py --trip-uuid TEST-2026-FULL
```

## Deploy

```bash
# Backend (Cloud Run)
make deploy-backend

# Frontend (Netlify)
cd frontend && netlify deploy --prod --dir=dist

# Ou os dois juntos da raiz:
make deploy
```
